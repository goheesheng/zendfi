# ZendFi User Guide

Welcome to the ZendFi user guide. ZendFi is a non-liquidatable crypto lending platform on the Base network. These docs walk you through every aspect of the platform, from your first loan to managing positions at expiry.

## Quick Links

- **Start borrowing**: Navigate to `/app` and connect your wallet
- **Network**: Base (chain ID 8453) — ensure your wallet is on Base before interacting
- **Supported collateral**: ETH and cbBTC
- **Loan currency**: USDC

## Overview of the Interface

The ZendFi app has five main tabs:

- **Borrow** — Create a new loan request (default tab)
- **Loans** — View and manage active loans
- **History** — See completed and expired loans
- **Lend** — Browse and fill limit orders as a lender _(coming soon)_
- **Settings (gear icon)** — Configure your preferences
